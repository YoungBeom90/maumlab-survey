## 실행 환경설정 ( 순서대로 진행해주세요 ‼️ )

## Installation
> 도커 설치 후 실행 👉 https://www.docker.com/products/docker-desktop/
```html
터미널에서 pull 받은 폴더경로로 이동, 아래 명령어로 docker compose 파일 실행
```
```bash
$ docker-compose up
```
```bash
$ npm install
```

## Running the app
```bash
$ npm run start
```

## API Request (Postman)
```
Repository 내 maumlab.postman.json 파일 postman 에서 import 하여 테스트 진행
```
